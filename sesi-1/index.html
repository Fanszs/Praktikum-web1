HTML

    HTML

HTML adalah bahasa markup yang digunakan untuk membuat struktur dan konten halaman web. Dengan HTML, Anda dapat menambahkan elemen seperti teks, gambar, tautan, tabel, dan formulir. HTML menggunakan tag, seperti h1 untuk judul, p untuk paragraf, dan img untuk gambar, untuk menentukan jenis konten.

CSS

    CSS

CSS adalah bahasa yang digunakan untuk mendesain dan mengatur tata letak halaman web. Dengan CSS, Anda dapat mengubah warna, font, ukuran, jarak, dan posisi elemen HTML. CSS memungkinkan pengembang untuk membuat halaman web yang menarik dan responsif, dengan memisahkan konten dari presentasi.

Javascript

    JavaScript

JavaScript adalah bahasa pemrograman yang memungkinkan interaktivitas di halaman web. Dengan JavaScript, Anda dapat membuat elemen yang responsif terhadap tindakan pengguna, seperti klik, hover, atau input. JavaScript juga digunakan untuk mengakses dan memanipulasi elemen HTML dan CSS, serta untuk berkomunikasi dengan server, memungkinkan aplikasi web yang dinamis.
